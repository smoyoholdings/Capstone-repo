# Capstone React App

## 👤 Author
Sipho Moyo

## 📌 Project Description
This is a React application built using Vite. It demonstrates routing, reusable components, and basic state management.

## 🚀 Features
- Home page
- About page
- Navigation bar
- React Router for navigation

## 🛠️ Installation
1. Clone the repository:
   git clone https://github.com/SMOYOHOLDINGS/Capstone-repo.git

2. Navigate into the project:
   cd capstone-app

3. Install dependencies:
   npm install

4. Run the app:
   npm run dev
